The following elements should serve you well as an initial set to become familiar with:

* `background`
* `border`
* `color`
* `display`
* `font-family`
* `font-size`
* `font-style`
* `font-weight`
* `line-height`
* `list-style`
* `margin`
* `padding`
* `text-align`
* `text-decoration`
* `text-indent`
* `text-transform`
